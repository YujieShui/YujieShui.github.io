**printf  "格式字符串" 候选字符**

| 格式字符串 | 作用                        |
| ---------- | --------------------------- |
| %d         | 将整数转成十进制            |
| %f         | 将整数转成浮点数            |
| %u         | 十进制无符号整数            |
| %o         | 将整数转成八进制            |
| %c         | 将整数转成对应的 ASCII 字符 |
| %s         | 将整数转成字符串            |
| %x         | 整数转成小写十六进制        |
| %X         | 整数转成大写十六进制        |
| %p         | 输出                        |
| %%         | 输出百分比符号，不进行转换  |

**printf 中特殊规定的字符**

| 规定字符 | 作用                      |
| -------- | ------------------------- |
| \n       | 换行操作                  |
| \f       | 清屏并换页                |
| \r       | 回车                      |
| \t       | Tab 符                    |
| \xth     | 用 16 进制表示的 ASCII 码 |
